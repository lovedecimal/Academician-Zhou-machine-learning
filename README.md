周志华《机器学习》经典算法复现

1. 项目简介
基于周志华《机器学习》（西瓜书），复现**C4.5决策树**（第4章）与**AdaBoost集成学习**（第8章）经典算法，以鸢尾花数据集为案例完成分类任务，并对比算法性能。

2. 核心功能
1. **C4.5决策树复现**  
   - 基于信息熵准则构建决策树。
2. **AdaBoost集成学习复现**  
   - 实现多决策树集成，提升分类精度。
3. **算法性能量化对比**  
   - 输出分类准确率与分类报告。
4. **可视化性能对比**  
   - 生成算法性能对比图表。

3. 环境安装
pip install -r requirements.txt

4. 快速运行
python main.py
